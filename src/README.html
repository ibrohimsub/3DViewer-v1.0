Introduction

Welcome to 3DViewer_v1.0, a 3D model viewer developed by Alphas Group. This program allows you to visualize 3D models, make various transformations, and customize the rendering of your models.

Installation

To install the 3DViewer_v1.0 program, follow these steps:

  1. Clone the repository from the Group's GitLab or GitHub.

  2. Open a terminal and navigate to the project directory.

  3. Build the program and generate the executable by running the following command: 

          make install>

  4. This will create an 3DViewer.app executable in the build directory.

  5. To run the program, execute the following command:

          make open_exe

Usage

Once you've installed the program, here's how to use it:

  1. Open the 3DViewer by running the executable.
  2. You can interact with the 3D model using buttons, sliders, and customization options in the graphical user interface.

Makefile Targets

Our Makefile provides several targets for building, testing, and distributing the program:

  • make install: Builds the program and generates the executable.
  • make uninstall: Removes the program and its artifacts.
  • make open_exe: Opens the program.
  • make dvi: Opens the documentation in your default web browser.
  • make dist: Creates a distributable package of the program.
  • make tests: Runs unit tests.
  • make gcov_report: Generates a code coverage report.
  • make valgrind: Runs the program with Valgrind for memory analysis.
  • make clean: Cleans up build artifacts and coverage reports.
  • make clang: Formats the source code using ClangFormat.

Customization

The program offers several customization options, including:

  • Sliders: You can use sliders to perform translations, rotations, and scaling on the 3D model.
  • Buttons: The program includes buttons for performing specific transformations (e.g., rotating around the X, Y, or Z axis) and for exporting the model.
  • Color Selection: Customize the color of edges and vertices.

About the Program

3DViewer_v1.0 is a 3D model viewer that provides tools for interactive exploration and manipulation of 3D models. It offers the following features:

  • Translation: Move the 3D model along the X, Y, or Z axis using sliders.
  • Rotation: Rotate the 3D model around the X, Y, or Z axis.
  • Scaling: Scale the 3D model up or down.
  • Edge and Vertex Customization: Choose colors for edges and vertices to enhance the visual representation.
  • Export: Export the 3D model as an image (BMP) or a GIF.


Alphas Group has developed this program to provide a user-friendly interface for working with 3D models, making it accessible and efficient for various use cases. Enjoy exploring your 3D creations with 3DViewer_v1.0!
